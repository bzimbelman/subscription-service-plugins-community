# Security policy

If you believe you've found a security issue in this repository (a community plugin that mishandles data, leaks credentials, or otherwise has security implications), please report it privately via [GitHub's security advisory feature](https://docs.github.com/en/code-security/security-advisories/guidance-on-reporting-and-writing-information-about-vulnerabilities/privately-reporting-a-security-vulnerability) for this repository.

Please do NOT file a public GitHub issue for security-sensitive findings.

We will acknowledge receipt within 5 business days.

For general bugs that are NOT security-sensitive, file a regular GitHub issue using the bug-report template.
